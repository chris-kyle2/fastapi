Performant Static Website on AWS With Scalable Notification System

A scalable notification system with real-time push notifications, scheduled notifications, and customizable preferences, powered by Serverless (AWS Lambda), AWS SNS, SQS, RDS(PostgreSQL), CloudFront, CloudWatch, AWS WAF, S3, Route 53 and EventBridge.

Tech Stack:

  • Python
  • FastAPI
  • AWS RDS
  • SNS
  • SQS
  • Lambda
  • EventBridge
  • SES
  • CloudFront
  • CloudWatch
  • AWS WAF
  • S3
  • Route 53

Key Features:

  • Web Push Notification:

    In Browser Notification Using pywebpush

  • Multi-channel notifications:

    Sends alerts via SMS, email (AWS SES), and Slack

  • Real-time push notifications:

    Uses AWS SNS and SQS for instant user updates

  • Scheduled notifications:

    Implements AWS EventBridge for milestone reminders, likes, and promotional messages

  • Customizable preferences:

    Users can choose their preferred notification channels

  • Dead Letter Queue:

    Handles undelivered messages with a dead letter queue
